AVHE

Required Software

AVHE should work on Windows and Mac machines, but only Windows has been tested at this time. Regardless of which Operating System used, a CAC reader is needed.

Your computer will need 2 pieces of software: Citrix Workspace and InstallRoot. If not already installed, they can be downloaded from the AVHE Support Page and only using Internet Explorer (once software is installed, any browser will work to access AVHE desktop).

When prompted, use your ID cert, enter PIN and after reading DoD Disclaimer, select the 'I accept' button. Towards the bottom of the page, these two software packages are found under the 'For End-Users' section.

Windows

Download Citrix Workspace and within the zip file, select Citrix WorkspaceApp to install. Download InstallRoot and within the zip file is the install file. Process through the default prompts. Once the console window appears, select 'Install Certificates.' Once installed, there should be a summary stating install is complete.

Mac

Awaiting eval from user then will update this section.

*** In order for software to initialize, a system restart is required regardless of Operating System. ***

Troubleshooting

If there are issues connecting, the DISA FBCA Cross-Certificate Removal Tool may need to be installed (For End User section). In some cases, the ActiveClient software may need to be installed.

Connecting

If Citrix Workspace and InstallRoot software have already been installed, from any Internet browser, go to AVHE and use your Authentication Certificate (email Certificate does not work).

Setup

Once the blue screens have processed, the virtual desktop will appear. The virtual desktop should be expanded to max to save confusion (because if not, there will be a task bar for the host computer & a taskbar for the virtual AVHE desktop.)

You will notice all applicable medical applications will be on the desktop & some bonus software such as MS Project, MS Publisher, MS Visio, as well as the common Office software (Outlook, Word, PowerPoint & Access).

Setup Outlook

Outlook should be setup if you want to send & receive encrypted email. To setup Outlook for the first time:

  • Open Outlook. In the upper left corner select File > Options
  • General tab < Personalize section > replace initials with your initials.
  • Mail tab > check the 'Always check spelling efore sending.'
  • Trust Center tab > Trust Center Settings... A new popup should open
  • Email Security > check 'Add digital signature to outgoing messages'
  • Within that same Email Security tab > Settings...
  • Signing Certificate Hash Algorithm change from SHA1 to SHA512.
  • Select OK > Publish to GAL > 2 pop up boxes proceed, hit ok. Close settings.

Find your Personal Network folder (H: drive)

Your personal network folder should appear in File Explorer. This is where you will find your Desktop and Documents.

In some cases, a shortcut for personal network drive or office network drives, can be mapped or a shortcut can be placed on your virtual desktop. Click This link to open that shortcut while on AVHE.

Map Office Network Drives (if needed)

To map the office network drives, open Windows Explorer (folder icon).

Access Issues & Quick Resolutions

Citrix Workspace & InstallRoot have to be installed on the computer. Sometimes when errors happen and connection isn't consistent, resetting the AVHE user account is a good solution.

  • AVHE website
  • Desktops tab
  • Details
  • Restart

For additional issues, we reccomend opening a ticket with the GSC for further troubleshooting.

There are multiple ways to create a ticket.

  • Call 1-800-600-9332 (CONUS), or by using the appropriate country code for OCONUS.
  • Create a ticket at the GSC website

To log a support ticket, please search 'AVHE' at the Remedy Center Request Management module.

Regardless of whether you call or email DHAGSC, please include the following information within the ticket:

  1. Name; including middle initial and previous last name if changed recently
  2. Current email address
  3. Contact phone number, include cell if you telework frequently
  4. Screen captures of any error messages
  5. Description of the problem to your best knowledge
  6. Name of other user(s) experiencing similar problems
  7. Version number of the following:
    • Windows 10 (type 'WINVER' from a command prompt). Optional, but helpful
    • Citrix Receiver or Citrix Workspace App
    • InstallRoot
    • ActiveClient (example - 7.1.0.210 or later is preferred). Again, not required but helpful.

Account Creation

You should have an AVHE account already, try logging in first. If you have account issues, follow the directions below:

Email DHA GSC for account creation and include the following:
  • Full name
  • EDIPI (located on back of CAC, a.k.a. DOD ID)
  • Email (your mail.mil)
  • Callback number
  • Location: U.S. Air Force Academy
  • Site Access: USAFA and Ft. Carson
  • What access: (choose only those that apply to you):
    • AHLTA
    • CHCS
    • DHA Desktop
    • DMLSS
    • Essentris
    • ADLS GoLearn

An incident number (8 digits) should be sent to your email.
