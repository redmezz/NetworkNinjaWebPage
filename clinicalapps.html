AHLTA Training

If you need to receive training for JKO AHLTA Live classes, please follow these directions:

  • Click on the Start Button
  • Right click on 'This PC'
  • Go down to 'More' and select 'Properties'
  • Take note of your computer name (starts with XQPZ...)
  • Install Firefox via the Software Center if it's not already installed.
    • Select Start, or the magnifying glass at the bottom taskbar to search for programs.
    • Search 'Software Center' and open the program
    • Find Firefox and install
